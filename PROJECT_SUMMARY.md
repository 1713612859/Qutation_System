## 技术实现亮点

1. **业务逻辑完善**
   - 报价单支持从套餐快速创建
   - 自动计算行小计、税额、总价
   - 支持百分比折扣和金额折扣

2. **状态流转管理**
   - 报价单状态：草稿 → 提交 → 审批/拒绝
   - 审批记录自动追踪

3. **用户体验**
   - 响应式设计，支持PC/平板/手机
   - Element Plus组件库，界面美观
   - 实时计算报价总价

4. **代码质量**
   - 统一使用Lombok简化代码
   - Service层事务管理
   - 统一的API响应格式

## 数据库表结构

1. `users` - 用户表
2. `customers` - 客户表
3. `products` - 产品表
4. `packages` - 套餐表
5. `package_items` - 套餐明细表
6. `quotations` - 报价单表
7. `quotation_items` - 报价明细表
8. `approval_records` - 审批记录表
9. `attachments` - 附件表
10. `system_settings` - 系统设置表

## 使用说明

1. **数据库初始化**
   ```bash
   mysql -u root -p quotation < backend/src/main/resources/schema.sql
   ```

2. **后端启动**
   ```bash
   cd backend
   mvn spring-boot:run
   ```

3. **前端启动**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **默认登录**
   - 用户名: admin
   - 密码: admin123
